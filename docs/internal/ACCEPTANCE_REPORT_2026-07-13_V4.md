## 实现细节

### 任务1: 交易联动余额
- `_update_account_balance()` 函数在 main.py 中
- 创建交易 → 自动扣减/增加账户余额
- 更新交易 → 先回滚旧值，再应用新值
- 删除交易 → 回滚余额
- `/parse` 和 `/webhook/notify` 端点均已集成

### 任务2: 投资持仓同步
- `create_position()` → 自动在资产表创建 `[持仓] XXX` 条目
- `update_position()` → 自动更新资产表的 current_value
- `close_position()` → 自动关闭资产表条目
- `add_trade()` → 买卖后自动更新资产

### 任务3: 白名单过滤器
- 三条规则必须同时满足：来源白名单 + 动作白名单 + 明确金额
- 来源覆盖：微信/支付宝/银行/电商/生活服务/信贷等
- 动作覆盖：扣款/收入/付款/支付/转账/还款/退款/消费等
- 金额正则：`[¥￥]?\s*(\d+)\s*元`
