# SilentBook V2 规划文档

## 一、V1 现状总结

### 已完成功能
- ✅ 通知解析器（5大平台，38个测试）
- ✅ 后端 API（交易、资产、负债 CRUD）
- ✅ 前端页面（8个页面）
- ✅ Docker 部署（5个容器）
- ✅ AI 分析（对接 OpenClaw agent）
- ✅ 通知推送（微信/飞书）
- ✅ 数据可视化（图表、报表）
- ✅ 用户认证（基础 JWT）
- ✅ 数据导入导出（CSV）

### 当前问题
1. **用户认证不完善** - 只有基础 auth.vue，缺少完整注册/登录流程
2. **服务器部署未完成** - 正在构建镜像，Nginx 配置待完成
3. **前端体验待优化** - Markdown 渲染、响应式布局需改进
4. **缺少用户引导** - 首次使用流程不完善
5. **缺少预算管理** - 预算设置和预警功能未实现

---

## 二、V2 目标

### 核心目标
1. **完善用户体验** - 从"能用"到"好用"
2. **提升产品稳定性** - 修复已知问题，完善测试
3. **增加实用功能** - 预算管理、多账户、数据同步
4. **优化性能** - 响应速度、资源占用

### 3.3 🆕 财务健康评分（P0.5，从 P2 提升，墨砚+远瞻共识）

#### 五维度评分模型（满分 100 分）
- [ ] 💰 **储蓄能力（25 分）**
  - 月储蓄率 ≥ 30% → 25 分
  - 月储蓄率 20-30% → 20 分
  - 月储蓄率 10-20% → 15 分
  - 月储蓄率 0-10% → 8 分
  - 月储蓄率 < 0% → 0 分

- [ ] 🛡️ **抗风险能力（25 分）**
  - 应急储备 ≥ 6 个月 → 25 分
  - 应急储备 3-6 个月 → 20 分
  - 应急储备 1-3 个月 → 12 分
  - 应急储备 < 1 个月 → 5 分
  - 无应急储备 → 0 分

- [ ] 📊 **预算纪律（20 分）**
  - 预算执行率 90-100% → 20 分
  - 预算执行率 80-90% → 15 分
  - 预算执行率 70-80% → 10 分
  - 预算执行率 60-70% → 5 分
  - 预算执行率 < 60% → 0 分

- [ ] 🏗️ **支出结构（15 分）**
  - 必要支出占比 < 50% → 15 分
  - 必要支出占比 50-60% → 12 分
  - 必要支出占比 60-70% → 8 分
  - 必要支出占比 70-80% → 4 分
  - 必要支出占比 > 80% → 0 分

- [ ] 📈 **投资增长（15 分）**（远瞻建议增加）
  - 投资收益月增长 ≥ 5% → 15 分
  - 投资收益月增长 2-5% → 12 分
  - 投资收益月增长 0-2% → 8 分
  - 投资收益持平 → 4 分
  - 投资收益下降 → 0 分

**评分等级：**
- 90-100：🏆 财务优秀
- 75-89：✅ 财务健康
- 60-74：⚠️ 财务一般
- 40-59：🔶 财务紧张
- 0-39：🔴 财务危险

**关键设计：**
- [ ] 每个维度给出具体改善建议
- [ ] 评分变化趋势比绝对值更重要
- [ ] 允许用户自定义权重
- [ ] 雷达图展示（远瞻建议）
